# forfun — 攻城游戏路径优化

对一张领地攻城地图做的数字化 + 攻防策略分析（纯属娱乐）。

## 内容

| 文件 | 说明 |
|---|---|
| `策略分析.html` | 主报告：分数结构、咽喉分析、协商/全力两方案、我方路线推荐（自包含，直接浏览器打开） |
| `rules.yaml` | 游戏规则单一来源：积分、10城上限、开放节奏、怪物难度 |
| `Map/Map.jpg` | 原始地图截图 |
| `Map/map_data.json` | 数字化地图：85 区块（等级/坐标/怪物/难度）+ 邻接边（已人工校对） |
| `Map/monster_types.json` | 区块 → 怪物类型 + 类型难度 |
| `Map/map_editor.html` | 邻接校对工具：拖节点、点两城加删边、导出 JSON |
| `Map/怪物分类表.png` | 怪物分类速查表 |
| `Map/scripts/` | 分析管线（Python） |

## 分析管线

```
rules.yaml + Map/map_data.json
  → plan2.py        # 分区、每周最优持有集、推进路线（城池分优先，同分选易怪）
  → render.py       # 分区图 / 咽喉图 / 锁死图 / 每周持有图
  → render_route.py # 我方推进路线图
  → build_report3.py# 汇总为 策略分析.html
```

依赖：`pip install numpy pillow scipy scikit-image pyyaml`。
改规则或地图后按上述顺序重跑即可。

## 核心结论（速览）

- 分数结构后期压倒一切：一座7级城一天 = 前期一整周。
- 8-1 仅能从三座 7 级城进入；7 级环互通，突破一点可横扫。
- 9 城锁死组合 {6-1..6-4, 5-8, 5-9, 7-1..7-3} ≤ 10 城上限，可完全垄断 7 级与 8-1。
- 5-8（暴食,易打）是全图性价比最高的卡位点：第 4 周开放，直通 6-1/6-4/7-3。
- 怪物难度东易西难（难档怪西 20 东 10），分边谈判的重要筹码。
